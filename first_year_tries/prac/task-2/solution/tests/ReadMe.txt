-- Это файл Readme для тестирования задания-2 --

Перед выполнением выполнить следующие команды в терминале:

cd <путь к каталогу с программой и тестами>
fpc ./2

Для запуска тестов выполнить следующую команду:
./2<tests/имя_теста имя_файла с графом(или путь к нему)

Всего 5 тестов: Каждый показывает выполнение одного режима работы программы.
В каждом тесте в строчках после слова "exit" указаны правильный вывод программы после ее исполнения.

Режимы 1,2,4,5 выполнены модификацией алгоритма Дейкстры, 3 - алгоритмом Поиска в Ширину.
